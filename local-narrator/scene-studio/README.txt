PANGEA SCENE STUDIO

Open the application

Double-click “Open Pangea Scene Studio.command” in the local-narrator folder.
The application runs only on this Mac and opens in your web browser. Closing the
Terminal window that opens with it stops the application.

Scene workflow

1. Choose a chapter and scene in the left column.
2. Edit the scene text. “Save new script version” always makes another version;
   it never overwrites the original scene file.
3. Use “AI identify all speakers.” A local language model reads the complete scene,
   assigns explicit and implied speakers, and accepts high-confidence lines. Review
   only the uncertain assignments. Add delivery notes only when desired.
4. Build a Character & Voice Bible. Give each synthetic voice a register,
   resonance, texture, pace, cadence, diction, accent, emotional range, vocal
   mannerisms, intensity rules, pronunciation notes, and traits to avoid.
5. Listen to and approve the narration-only Draft.
6. Plan sound cues, record their source and license, then approve the narration
   plus sound-effects mix.
7. Add restrained background music only after the sound-effects version is
   approved, then approve the final layered mix.
8. Use Compare to hear Draft, With Sound Effects, and With Music Background.
9. Add an intro and exit for episode releases, or leave both off when compiling
   the whole book.

Important safeguards

- Manuscript and audio assets stay local unless you deliberately choose a file
  for an online review copy later.
- Speaker suggestions require review. Ambiguous lines remain unassigned.
- Every character voice is book-specific and should be approved before rendering.
- Do not imitate or clone a real person unless an authorized reference is supplied
  and permission is explicitly confirmed.
- Keep narration, sound effects, ambience, and music as separate masters. The
  application tracks approvals without replacing earlier versions.
